Instructions

  • Select a monster from the list on the left to view its stats.
  • The Monster List can be sorted either alphabetically or reverse alphabetically, and it can be searched using the searchbar and hitting enter.
  • To save monsters for future easy use, like an encounter, click on the monster and then click Save Current Selection.
  • You can view this list by clicking Show Past Selections and go back by clicking Show Main List.
  • To Roll, set up the number of dice, the number of sides per die, and the modifier then click Roll.
  • UI settings will be saved for future use in local storage. To clear that, click Clear Local Storage
